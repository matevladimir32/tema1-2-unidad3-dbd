6. Lenguaje Formal de Consultas: Comando UPDATE

Contenido

Comando de SQL que permite modificar la información de una o más filas de una tabla. Este comando es capaz de modificar varios registros y varios campos a la vez.

La sintaxis simple o estructura de una consulta UPDATE de SQL es la siguiente:

UPDATE NombreTabla SET Columna = Valor

Pero cabe resaltar que esa es la estructura de una consulta UPDATE sencilla sin condiciones, que modificaría todos los registros de la tabla indicada.

La estructura completa de esta sentencia SQL es la siguiente:

UPDATE NombreTabla SET Columna = Valor WHERE CONDICION

Ejemplo:

Si se cuenta con la tabla ALUMNOS y los registros mostrados en la imagen anterior y se desea modificar o actualizar el campo de MARIA VERONICA, con la dirección ‘RESIDENCIAL LOS ROBLES CASA 5’, la sentencia a ejecutar seria la siguiente:

UPDATE ALUMNO SET Direccion= 'RESIDENCIAL LOS ROBLES CASA 5' where Alumnoid = 2

Pero si el caso fuera que además de la dirección se desea modificar los apellidos, la sentencia a ejecutar seria la siguiente:

UPDATE ALUMNO SET Apellido= 'VALENCIA DE TORRES', Direccion= 'RESIDENCIAL LOS ROBLES CASA 5' where Alumnoid = 2

Como se observa se están actualizando dos campos de un mismo registro. De igual manera podríamos actualizar todos los campos del registro exceptuando aquellos que sean calculados o llaves primarias. Pero con el comando se puede:

  • Actualizar una columna de uno o varios registros en concreto.
  • Actualizar una columna de todos los registros de una tabla.
  • Modificar varias columnas de una o más filas de una tabla.
  • Modificar varias columnas de todas las filas de una tabla.
